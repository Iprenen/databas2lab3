-------------------------------------------------------------------------------
DESCRIPTIONS
-------------------------------------------------------------------------------
This folder contains
- kd.jar : 
	KDTree implemented by Simon Levy

- lab3_stub.osql: 
        You should write AmosQL code here to complete the assignment

- KDTreeIndex_Stub.java
	Skeleton Java code. You should complete implementations of 
        foreign functions.

- winequalitysample.cvs
	Data set. You should not modify it, otherwise the assignment 
        validation script might fail

- readme.txt

-------------------------------------------------------------------------------
SET UP ASSIGNMENT'S SKELETON CODE
-------------------------------------------------------------------------------
1) Download sa.amos to the root directory of your USB stick. Unzip it

2) Run (or click on) assignment3.cmd

-------------------------------------------------------------------------------
TASK
-------------------------------------------------------------------------------
The assignment requires you to
 - complete foreign functions in Java. See file KDTreeIndex_Stub.java
 - complete all TODO(s) in lab3_stub.osql 


Compile KDTreeIndex_Stub.java
	javac KDTreeIndex_Stub.java

Run sa.amos
	sa.amos

When you are in sa.amos you get the prompt:
        - [sa.amos] 1> enable_java();
	- Copy and run single AMOSQL queries
